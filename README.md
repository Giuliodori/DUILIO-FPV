# DUILIO FPV — Limitless Remote Control

> Real-time remote control. No distance limits.

Duilio FPV unlocks real-time remote control with ultra-low latency, allowing you to drive and operate systems anywhere — without distance limits.

Leveraging WiFi mesh and 4G/5G networks, it delivers smooth, high-quality control far beyond traditional radio systems.

Flexible by design and hardware-agnostic, Duilio FPV supports multiple motor control methods (PWM, step/dir, analog, and more). It achieves maximum performance, safety, and reliability when integrated with [Duilio F4 hardware](https://github.com/Giuliodori/duilio_f4), where everything is simpler, fully tested, and ready to use.

## Project Goal
>>>>>>> b78399e (aggiornato readme)

Duilio FPV was born from a real need: enabling a person with a physical disability to mow the lawn remotely.

The vision is simple but powerful — bringing physical actions into the digital world, so anyone can operate machines and perform real tasks from anywhere.

From this starting point, the possibilities expand to robotics, remote vehicles, automation, and many other real-world applications.
