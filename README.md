# DUILIO FPV

Duilio FPV is a modular ecosystem for remote control systems, robotics, and distributed motor control.

## Architecture

The project is divided into:

- **Onboard (Raspberry Pi)**  
  Control, communication, sensors, and low-level logic  
  https://github.com/giuliodori/duilio_fpv_onboard

- **Offboard (PC)**  
  UI, control interface, remote management  
  https://github.com/giuliodori/duilio_fpv_offboard

## Goal

Create a flexible and scalable platform to control vehicles, robots, and automation systems using Duilio hardware.
