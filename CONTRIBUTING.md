# Contributing

This repository is the public hub for the Duilio FPV ecosystem.

## What Belongs Here

Use this repository for:

- README and documentation improvements related to the ecosystem overview
- hub-level issues
- public-facing project presentation updates
- links, structure, and navigation improvements between repositories

## What Does Not Belong Here

Open code-specific issues and pull requests in the relevant repository:

- `duilio_fpv_onboard` for onboard software
- `duilio_fpv_offboard` for offboard software
- `duilio_f4` for hardware-related work

## Before Opening A Pull Request

- confirm that the change belongs to this hub repository
- keep documentation concise and accurate
- avoid duplicating detailed technical documentation that should live in subproject repositories
- preserve this repository as a clean entry point for the ecosystem

## Pull Request Scope

Good pull requests for this repository are usually small and focused:

- fix or improve project overview text
- clarify navigation between repositories
- update external links
- improve contributor guidance

## Questions

If you are unsure where a change belongs, open an issue here first and describe the proposed change briefly.
